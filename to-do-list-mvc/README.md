ToDo List - MVC version

- MVC model utilised.
- Store tasks in a MySQL database.
- Add, mark as completed, and delete tasks. (CRUD model demo)
- Use of AJAX for a better user experience.
- Style and pleasant basic UX/UI with tailwind css.
- Task limit = 20.
